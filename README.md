# Loader

Loader is a simple Python utility for loading and managing:
- sprites
- JSON files
- INI files

It supports threading for faster loading of resources.

---

## Requirements

- Python 3.10+
- Poetry

---

## Installation

Clone the repository:

```bash
git clone https://github.com/mikolaj09/loader.git
cd loader
poetry install
```

---

## Usage

```python
from loader import Loader

loader = Loader()
```
